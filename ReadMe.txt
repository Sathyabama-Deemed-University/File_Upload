Note:
Api return base64 encoded bytes data for excel files
Default datatype formats:
    int-Int64
    float-Float64
    Date-Date
    string-Utf8
    Date format:
    %d - day
    %m - month
    %Y - year
    Example:
    %d-%m-%Y
    %d/%m/%Y
 For Reference for datatypes -> https://pola-rs.github.io/polars/py-polars/html/reference/datatypes.html
